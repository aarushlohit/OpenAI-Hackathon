# Gemini Operating Guide

Gemini is expected to help with broad reasoning, documentation review, and multimodal design decisions while respecting the repository boundaries.

Use Gemini-style analysis for:

- Cross-checking scam investigation workflows.
- Reviewing threat models for recruitment fraud.
- Proposing evidence taxonomies.
- Auditing documentation for contradictions.

Do not bypass the gateway layer or introduce provider-specific behavior inside agents.
